# Code of Conduct

✅ Be Respectful  
- Treat all contributors with courtesy and empathy.  
- Avoid personal attacks, offensive language, or discriminatory remarks.

✅ Collaborate Constructively  
- Offer feedback with kindness and clarity.  
- Focus on the code and ideas—not the person.

✅ We’re All Learning  
- Mistakes happen. Be open to feedback and willing to grow.  
- Disagreements are okay—keep them civil and solution-focused.
